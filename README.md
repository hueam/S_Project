## 사용법!
1. 서버열기
   1. 다운받는다
   2. 서버를 연다
2. 클라이언트
   1. 다운 받는다!
   2. 서버를 실행한 사람의 IP를 다른 사람에게 알려준다!
   3. 제데로 입력후 Start!
   4. 한명은 방이름을 정하고 제작
   5. 다른 사람들은 같은 방이름 쓰고 참여!
   6. 모두가 Ready하면 시작!
